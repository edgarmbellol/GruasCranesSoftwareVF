Información sobre Datos Maestros
¿Qué son los datos maestros?

Los datos maestros son la información de configuración base del sistema que se utiliza en diferentes módulos. Es importante mantener esta información actualizada para el correcto funcionamiento del sistema.

Recomendaciones:
  • No elimines datos que estén siendo utilizados
  • Mantén la información consistente
  • Revisa regularmente la configuración
  • Haz respaldos antes de cambios importantes
